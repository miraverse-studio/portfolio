# Prompt Launcher v1

> 一言: Claude Code 用プロンプトを差し込み生成・コピーするツール
> 種別: 動くツール ／ ランク: A（コピー・保存・復元 確認済み）

## これは何か
開始プロンプトをテンプレートから生成しワンクリックでコピー。入力は localStorage に保存・復元。
クリップボード不可環境向けのフォールバックUIあり。

## 触り方
`../apps/prompt-launcher/index.html` を開く → 入力 → プレビュー更新 → コピー → 別アプリに貼付

## 見てほしいポイント
- クリップボードAPIのフォールバック設計
- Windowsパスのバックスラッシュを壊さない処理
- localStorage キー設計

## 既知の弱点
- 単機能ツール（範囲を意図的に小さく保つ）
