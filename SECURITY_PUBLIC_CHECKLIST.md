# 公開前セキュリティチェックリスト

> このポートフォリオを一般公開する前に、全項目を満たすこと。
> 全項目 PASS が「一般公開してよい」唯一の条件。

## A. 秘密情報
- [ ] APIキーがない
- [ ] .env がない
- [ ] token / secret / password 文字列がない
- [ ] 鍵・認証情報の一覧ファイルを含めていない

## B. 個人情報
- [ ] メールアドレス・電話番号がない
- [ ] 本名・職場名・友人名がない

## C. ローカル環境情報
- [ ] ローカルのフルパス（C:\Users\<ユーザー名>\... 形式）がない
- [ ] クラウド同期フォルダのフルパスがない
- [ ] PC名・OSユーザー名がない

## D. ログ・内部資料
- [ ] AI開発ツールのログ全文がない
- [ ] 内部プロンプト全文がない
- [ ] セッションログ（*.jsonl 等）がない
- [ ] 内部設計フォルダ（_design/）を公開対象に含めていない

## E. スクリーンショット
- [ ] 画像内に個人情報・パス・トークン・実名の写り込みがない

## F. Git
- [ ] 親リポジトリから独立した単独リポジトリである
- [ ] Git履歴（過去コミット）に秘密情報がない
- [ ] commit author 名／email がハンドル運用と矛盾しない

## G. 内容の正直さ
- [ ] 未確認の成果物を完成品と書いていない
- [ ] 一般公開してよい範囲だけになっている

## サインオフ
全 PASS → 友人へ private 共有可。一般公開はさらに段階を分ける。
